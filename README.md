# Portfolio Website built in React

### k.d. Bromley

A React app built with Vite, using JSX, JS, HTML and CSS. 

A portfolio website tailored for my current job search, as I return to Web Development after two years away spent working as a wrangler on trail riding ranches in California, Wyoming, and Arizona. This site showcases my professional experience in Web Development, first and foremost, with the most significant projects I completed during my two years as a Junior Developer at Plucky (Burbank, CA) and freelance gigs in the interim. It also details the role my "break" from web development played, and the roles and valuable experiences I had as a wrangler. Finally, a short portfolio of my creative writing work, which is a creative outlet as much as it is a professional hobby and a skillset that I value.

The site itself serves as a preview of my skills in producing clean, focused code, modularity, consistency, and adhering to best practices, despite the few years I spent out of the industry.
